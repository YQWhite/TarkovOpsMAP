# TarkovOpsMap - 塔科夫战术地图同步系统

TarkovOpsMap 是一款专为《逃离塔科夫》玩家设计的实时战术地图同步工具。它支持多端同步绘图、地图切换以及多语言显示，非常适合小队作战时进行战术规划和位置标注，使用了AI辅助编写。

## 🌟 功能介绍

* **实时同步**：基于 WebSocket 技术，所有连接的队友都能实时看到你的绘图标注和地图切换。
* **多端支持**：针对移动端（手机/平板）进行了深度优化，修复了画布坍塌和交互冲突。
* **动态地图加载**：自动读取 `maps` 文件夹下的子目录及图片，无需手动修改代码即可添加新地图。
* **战术工具箱**：包含画笔、橡皮擦、自由平移、一键清屏等核心功能。
* **多语言适配**：支持中文和英文切换，且支持外部 `locales.js` 自定义翻译。
* **便捷部署**：支持使用 `pkg` 打包为独立 `.exe` 文件，无需安装 Node.js 即可运行。

## 🛠 如何构建 (Build)

如果你想自行打包成可执行文件，请遵循以下步骤：

1. **环境准备**：确保你的电脑安装了 [Node.js](https://nodejs.org/)。
2. **安装依赖**：
在项目根目录下打开终端，运行：
`npm install`
3. **安装打包工具**：
`npm install -g pkg`
4. **执行打包**：
运行以下命令将项目打包为 Windows 64位程序：
`pkg . --targets node16-win-x64 --out-path dist`
5. **整理产物**：
打包完成后，在 `dist` 文件夹下会生成 `server.exe`。

## 🚀 如何通过 EXE 启动

启动程序非常简单，但需要确保文件结构正确：

1. **准备运行目录**：
创建一个新文件夹，将生成的 `server.exe` 放入其中。
2. **放置必要资源**：
必须将以下文件/文件夹放在 `server.exe` 的**同一级目录下**：
* `maps/` 文件夹（里面放你的地图文件夹及图片）
* `locales.js` 文件（语言包）


3. **启动服务**：
* 双击运行 `server.exe`。
* 控制台会显示 `>>> 塔科夫战术地图服务器已启动!`。
* 默认访问地址为：`http://localhost:3000`。


4. **局域网访问**：
如果队友需要访问，请自建服务器并让队友使用服务器的公网ip连接。
